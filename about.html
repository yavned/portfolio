About Me

I am a Senior Full Stack Web developer with over seven years of experience. I have build and launched multiple projects from scratch, for start-ups, private companies and government organisations, carrying the development of their back-end and front-end codebases.

My current toolset includes .Net C# and Javascript with Angular, React and Vue.js frameworks.

BACK-END WEB DEVELOPMENT

My current stack involves C# and .Net Framework.

  • Lead development of web apps in .Net C# (.NET Core, ASP.NET MVC).
  • Building scalable, reliable and elastic cloud based RESTful API's with Azure functions, Azure storage, & Service Bus.
  • Using different types of databases (like MSSQL, MySQL, Oracle, PostgreSQL) dividing servers into different machine nodes / Docker containers.
  • Refactoring existing applications, by improving code readability, separating concerns into separate functions and classes, splitting out business logic from app's views and controllers.
  • Writing automated tests.

FRONT-END WEB DEVELOPMENT

I'm specialised in applications written in Angular. But also have a good hands-on experience with React and Vue.js.

  • Lead TypeScript/JavaScript development: bootstrapping or refactoring existing app architecture by improving extensibility and reliability.
  • Full integration of front-end development using tools like Webpack with features like automatic code reload, code minifications and multiple environment support.
  • Vast experience in many JS ecosystems, from using many libraries such as ngrx, RxJS, jQuery, PrimeNG, Datatables, ChartJS and many others.
